Completely Free. Forever.
Simply follow these steps:
  1. Set up a GitHub account if you haven't already
  2. Create a new GitHub Pages site for a project or for your user/organization. You can also use an existing GitHub Pages Site.
  3. Create a new CNAME file in the root directory of the repository and put the domain you'd like to use in the form
    YOURPROJECT.is-dummy-thi.cc or YOURUSER.is-dummy-thi.cc
  4. Wait up to 24 hours for your new domain to take effect
  5. Go to YOURPROJECT.is-dummy-thi.cc and ensure it's working. If it isn't, check your CNAME file or the GitHub repository settings.
  6. Profit! and make sure to share this service or maybe donate :)
